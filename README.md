# JavaProjetEseoE3e
Projet Java de gestion intelligente de tâches (STRMS) réalisé dans le cadre du semestre de printemps 2026 (E3e S6) à l'ESEO.
Description détaillée (à copier-coller dans ton fichier README.md) :Smart Task & Resource Management System (STRMS) 
Le STRMS est une application logicielle développée en Java permettant de simuler la gestion de tâches dans un environnement d'ingénierie. 
Ce projet met en pratique la programmation orientée objet (POO), les structures de données, ainsi que la gestion des erreurs.
✨ Fonctionnalités principales :
Gestion des utilisateurs : Attribution d'autorisations spécifiques selon trois rôles : Admin, Manager et Ingénieur.
Gestion des dépendances : Système de vérification empêchant le démarrage d'une tâche si ses prérequis ne sont pas validés, avec détection des dépendances circulaires.
Historique et Traçabilité : Enregistrement complet de toutes les actions et des changements d'état pour chaque tâche.
Priorisation : Exécution et tri des tâches en fonction de leur niveau de priorité (LOW, MEDIUM, HIGH, CRITICAL).
